_Project Name_: **DigiTools**

_Description_: DigiTools is a platform where you can find useful digital tools. You can browse products, add them to your cart, and use them to make your work easier and faster.

_Technologies I used_:

                      **•** React.js  
                      **•** Tailwind CSS  
                      **•** DaisyUI  
                      **•** JavaScript (ES6+)  
                      **•** React-Toastify (NPM Package)  
                      **•** JSON (for product data)

_3 Features about my project_:

                              **•** Dynamic Products Display  
                              **•** Interactive Cart System  
                              **•** Responsive Design
